3.1. Token Operations

The operations described in this section allow service developers to get and validate access tokens.

VerbURIDescription
POST/v1.0/tokens Authenticate a client and generate an access token.
GET/v1.0/tokens/{tokenId} Check that a token is valid and return the token details.
DELETE/v1.0/tokens/{tokenId} Revoke Token.
GET/v1.0/tokens/{tokenId}/applications/{applicationId} Check if token has access to an application.
GET/v1.0/tokens/{tokenId}/applications/{applicationId}/roles/{roleId} Check if token has role on an application.
